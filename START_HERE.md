## 🔑 What You Need

For **Starknet** and **Ethereum**, you only need:

### 1. Infura API Key (Free)
   - **Get it here:** https://infura.io
   - **What it does:** Connects you to Ethereum and Starknet networks
   - **Cost:** FREE (100,000 requests/day)
   - **Time:** 2 minutes to sign up

### 2. Optional API Keys
   - **Etherscan:** https://etherscan.io/apis (for contract verification)
   - **BSCScan:** https://bscscan.com/apis (for BSC verification)

## 🚀 Quick Steps

1. **Sign up at Infura:**
   ```
   Visit: https://infura.io
   Create account → Create project → Copy Project ID
   ```

2. **Run our setup tool:**
   ```bash
   npm run setup
   ```

3. **Test your setup:**
   ```bash
   npm run check:balance:eth
   ```

4. **You're done!** 🎉
